# :punch: phist :punch:

An implementation of Loom's Proof-of-History.


# Developing

Install Rust and Cargo:

```bash
$ curl https://sh.rustup.rs -sSf | sh
```

Download the source code:

```bash
$ git clone https://github.com/garious/phist.git
$ cd phist
```

Run the test suite:

```bash
cargo test
```
