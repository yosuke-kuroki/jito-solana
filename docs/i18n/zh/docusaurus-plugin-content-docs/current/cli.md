在本节中，我们将描述如何使用Solana命令行工具创建_钱包_，来发送和接收SOL代币以及通过委托质押来参与到集群中。

为了与Solana集群进行交互，我们需要使用其命令行界面，也称为CLI。 命令行是Solana核心团队首次部署新功能所用到的工具。 命令行界面不一定是最容易使用的，但它提供了对Solana帐户的最直接、灵活和安全的访问。
